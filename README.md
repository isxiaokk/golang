# golang

一个简单的 Go 入门示例项目：在终端打印 `Hello, Go!`。

## 环境要求

- [Go](https://go.dev/dl/) **1.22+**

## 快速开始

```bash
git clone git@github.com:isxiaokk/golang.git
cd golang
go run .
```

预期输出：

```text
Hello, Go!
```

## 项目结构

| 文件       | 说明           |
| ---------- | -------------- |
| `main.go`  | 程序入口       |
| `go.mod`   | Go 模块定义    |
| `README.md`| 项目说明       |

## 模块名

本仓库模块名为 `golong`（见 `go.mod`）。若你 fork 后想改名，可修改 `go.mod` 第一行的 `module` 声明。
